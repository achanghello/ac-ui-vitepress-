# Icon 图标

暂时只有一个加载图标,后续会提供了一套常用的图标集合。

## 使用方法1

直接通过设置类名为 ac-icon-iconName 来使用即可。例如：

:::demo

```vue
<template>
  <i class="ac-icon-jiazai"></i>
  <ac-button type="primary" icon="ac-icon-jiazai">加载</ac-button>
</template>
```

:::
## 使用方法2

使用标签`<ac-icon>`。例如：

:::demo

```vue
<template>
  <ac-icon name="jiazai"></ac-icon>
</template>
```

:::
