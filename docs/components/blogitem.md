# blogitem 博客单项

博客搜索的结果的每一项

## 基础用法

主页展示，搜索页展示

:::demo 

```vue
<template>
  <ac-blogitem></ac-blogitem>
</template>
```

:::

## 基础用法

博客单项的第二种形式，通过normal调节

:::demo 

```vue
<template>
  <ac-blogitem :normal="false"></ac-blogitem>
</template>
```

:::

## Attributes

|参数|说明|类型|可选值|默认值|
|-|-|-|-|-|
|normal|控制两种展示形式|boolean|—|true|
